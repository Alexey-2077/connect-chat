# Автоматическое обновление GitHub

Этот проект настроен для автоматического обновления изменений в GitHub.

## Варианты автоматизации

### 1. GitHub Actions (Рекомендуется) ⭐

Автоматически запускается на серверах GitHub каждый час.

**Как работает:**
- GitHub Actions проверяет изменения каждый час
- Если есть изменения, автоматически создает коммит и пушит их
- Не требует локального компьютера

**Настройка:**
1. Убедитесь, что файл `.github/workflows/auto-commit.yml` существует
2. GitHub Actions автоматически активируется при первом коммите
3. Проверьте статус в разделе "Actions" на GitHub

**Проверка:**
- Перейдите на GitHub → ваш репозиторий → вкладка "Actions"
- Убедитесь, что workflow "Auto Commit Changes" активен

### 2. Локальный скрипт (Windows)

Автоматически запускается на вашем компьютере каждый час.

**Настройка (один раз):**

1. Откройте PowerShell **от имени администратора**
2. Перейдите в папку проекта:
   ```powershell
   cd C:\loveconnect
   ```
3. Запустите скрипт настройки:
   ```powershell
   .\setup-auto-push.ps1
   ```

**Ручной запуск:**
```powershell
.\auto-push.ps1
```

**Проверка задачи:**
```powershell
Get-ScheduledTask -TaskName "GitHubAutoPush_ConnectChat"
```

**Удаление задачи:**
```powershell
Unregister-ScheduledTask -TaskName "GitHubAutoPush_ConnectChat" -Confirm:$false
```

## Как это работает

### GitHub Actions
- ✅ Работает даже когда компьютер выключен
- ✅ Не требует локальных ресурсов
- ✅ Автоматически активируется

### Локальный скрипт
- ✅ Работает локально
- ✅ Можно настроить индивидуально
- ⚠️ Требует, чтобы компьютер был включен

## Рекомендации

1. **Используйте GitHub Actions** - это самый надежный способ
2. Если нужен локальный контроль - используйте локальный скрипт
3. Можно использовать оба варианта одновременно

## Безопасность

- GitHub Actions использует токен репозитория (безопасно)
- Локальный скрипт работает только на вашем компьютере
- Все изменения логируются с временными метками

## Частота обновления

По умолчанию обновления происходят **каждый час**. 

Для изменения частоты:
- **GitHub Actions**: отредактируйте `.github/workflows/auto-commit.yml` (строка `cron: '0 * * * *'`)
- **Локальный скрипт**: отредактируйте `setup-auto-push.ps1` (строка `-RepetitionInterval (New-TimeSpan -Hours 1)`)

## Примеры cron для GitHub Actions

- Каждые 30 минут: `'*/30 * * * *'`
- Каждые 2 часа: `'0 */2 * * *'`
- Каждый день в 12:00: `'0 12 * * *'`
